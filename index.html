      Todoist sync

      Paste your personal API token from Todoist → Settings → Integrations → Developer. It is stored only on this device and is sent straight to Todoist — never to anyone else.

      Any Todoist filter works — combine with & (and) / | (or). Examples: today & @errand, #Work & p1, overdue | today. Imported tasks map to the OODA channel whose name matches their Todoist project (else the first channel).

      Places dated and recurring Todoist tasks onto their due day in the Day / Week / Month views. Note: Todoist only exposes a recurring task’s next occurrence, so a daily‑repeating task appears on its next due day, not literally every day — it rolls forward as you complete it.

      Import tasks from Todoist

      Pull in any task — leave the box blank for all active tasks, or type a filter to narrow it. Tick the ones to bring into today.

      Sync across devices

      Keeps your tasks, times and reflections in sync between phone and laptop using a private GitHub Gist — free and reliable. Create a GitHub personal access token with the gist scope (github.com/settings/tokens), paste it below, and Save. The token stays on this device; only your OODA data goes to the gist.
